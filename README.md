# hr-interview · HR 面试题设计工作台

> **10 分钟对话，拿走一份可直接开会用的面试打分表。**
> 告别手工出题、手工做 Excel——AI 主导五步流程，交付能力模型、行为题、情景题、STAR 追问和 1-5 分评分标准，最终生成本地 `.xlsx` 文件。**数据不出本机，不需要任何 API Key。**

---

## 你什么时候需要它？

| 场景 | 没有它 | 有了它 |
|---|---|---|
| 临时招一个新岗位 | 东拼西凑 2-4 小时出题 | 10 分钟对话，全套题目 + 打分表到手 |
| 面试官水平参差不齐 | 打分标准全靠个人感觉 | 每道题附 1-5 分可观察行为描述 |
| 面试完候选人说不清好不好 | 凭印象讨论，结果争半天 | 总分自动汇总，评价有据可查 |

---

## 它交付什么？

生成一份 **4 工作表 Excel 文件**，打开即用：

| 工作表 | 内容 |
|---|---|
| 📋 封面 | 岗位信息、题目总数、使用说明 |
| 🧩 能力模型 | 3-6 项核心能力 + 2-3 项关键特质，逐条说明 |
| 📝 面试题库 | 每维度 2 道题（行为 + 情景），含 STAR 追问和 1-5 分评分标准 |
| ✅ 打分表 | 面试官现场使用，橙色评分列，总分自动汇总 |

**样例文件**（可直接下载查看）：
👉 [`examples/高级产品经理-面试打分表-样例.xlsx`](examples/高级产品经理-面试打分表-样例.xlsx)

## 快速安装

```bash
npx skills add andy-he0101/my-claude-skills -g
```

**前置依赖**（仅生成 Excel 时需要）：

```bash
pip install openpyxl
```

没有 Python？也没关系——Skill 会自动降级为 Markdown 格式输出，你复制后手动填表即可。

## 触发方式

直接用自然语言说，无需记命令：

```
帮我设计面试题
我要面试一个高级产品经理
帮我做个 Java 后端工程师的面试评分表
帮我构建运营总监的能力模型
生成面试打分表
STAR 追问怎么设计
帮我准备明天的技术面试官培训素材
```

## 完整示例

**输入：**
```
我要面试一个「增长产品经理」，业务背景是电商 App，团队 5 人，
现在有 JD 但没有面试题，帮我出一套。
```

**执行过程（五步）：**
1. 收集岗位信息 → 确认
2. 生成岗位价值分析 → 用户确认
3. 构建能力模型（4 核心能力 + 2 关键特质）→ 用户确认
4. 生成 12 道面试题 + 评分标准 → 用户确认
5. STAR 追问拆解 → 生成本地 Excel

**输出：**

```
✅ Excel 打分表已生成！

📁 文件路径：C:\Users\xxx\Desktop\增长产品经理-面试打分表-20260612.xlsx

包含 4 个工作表：
• 封面 — 岗位信息概览
• 能力模型 — 4 项核心能力 + 2 项关键特质
• 面试题库 — 12 道面试题（含 STAR 追问 + 1-5 分评分标准）
• 打分表 — 面试官用评分表（橙色评分列，总分自动汇总）
```

## 安全边界

- ✅ 所有数据仅保存在本地，**不上传任何服务器**
- ✅ 生成文件前会确认保存路径，**不静默覆盖**已有文件
- ✅ 不执行任何网络请求
- ✅ Python 脚本执行失败时，**自动降级**为 Markdown 格式输出
- ❌ 不适用于批量导入简历或候选人数据库操作
